syslog-ng
=========

Installs syslog-ng with the option to push to a central syslog-ng server /
option to listen for messages from other syslog-ng installations.

Requirements
------------

None

Role Variables
--------------

This role does not have variables.

Dependencies
------------

None

Example Playbook
----------------

Usage is as simple as:

    - hosts: servers
      roles:
         - syslog-ng
